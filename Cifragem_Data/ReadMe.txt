Para cifrar digite no terminal: 
python diretorio/do/programa/cifrar.py 18/12/46 diretorio/do/arquivo/mensagemEmClaro.txt diretorio/do/arquivo/mensagemCifrada.sec
Obs.: 18/12/46 é uma data fictícia utilizada na cifragem. Observe que o ano deve possuir apenas os dois últimos dígitos.

Para decifrar digite no terminal:
python diretorio/do/programa/decifrar.py 18/12/46 diretorio/do/arquivo/mensagemCifrada.sec
